# 🧸 Toy Heaven

## 🌟 Project Overview

**Toy Heaven A9** is a modern, responsive web application designed to showcase and manage a collection of toys. It provides users with a smooth browsing experience, authentication features, and interactive UI elements to explore toys efficiently.

---

## 🎯 Purpose

The purpose of this project is to:

* Build a fully functional React-based web application
* Practice authentication and routing using Firebase
* Create a visually appealing UI with Tailwind CSS and DaisyUI
* Enhance user experience with animations, loaders, and notifications

## ✨ Key Features

* 🔐 **Authentication System** (Login/Register with Firebase)
* 🧸 **Toy Listings & Details View**
* 🔎 **Dynamic Routing with React Router**
* ⚡ **Fast Performance with Vite**
* 🎨 **Modern UI with Tailwind CSS & DaisyUI**
* 🔔 **Toast Notifications for User Feedback**
* ⏳ **Loading Indicators & Spinners**
* 📱 **Fully Responsive Design**
* 🧭 **SEO Optimization using React Helmet**
